# Repos And Work Surfaces

Default intake path: canonical work-item folder under `work-items/<id>/`.

Recommended work-item structure:

```text
work-items/<id>/
  README.md
  source/
  metadata/
  analysis/
  outputs/
  decisions.md
  next-actions.md
```

Use issues or task cards when collaborators need a no-terminal upload path or a
clear next-step assignment.
