# Opportunity Planning Refresh

Refresh deadlines, conferences, grants, renewals, launches, procurement windows,
or other operator-relevant opportunities.

Rules:

1. Verify current dates from official sources before posting.
2. Connect each opportunity to work-item readiness and owner fit.
3. Mark closed, not posted, or unverified dates explicitly.
4. Write a compact table with opportunity, fit, deadline/status, location, and
   URL when available.
5. Update memory only when priorities or verified dates change.
